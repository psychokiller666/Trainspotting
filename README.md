uikiller，它是CocosCreator下的UI编程库，用是使用**名命规则**来指挥节点、组件和触摸事件，从而大量减少冗余代码量与编辑器上的设置。
他的前身是工作在cocos-js下，名为UILoader你可以通过从下面链接获取
https://github.com/ShawnZhang2015/UILoader。

我为uikiller写了一篇使用教程，请从下在连接获取，如果对你有所帮助，我会感到很开心！
